# 📱 网站内容快速更新手册 (Operator Guide)

本手册专为配合社交媒体（TikTok/IG/小红书）内容更新而设计，确保用户从视频点击进入后能立即实现“即见即买”。

## 1. 更新 Hero 视频（首页置顶大片）
为了保持视觉一致性，建议每次发布热门视频后更新此处。

**文件路径：** `src/components/HeroVideo.tsx`

**操作步骤：**
1. 将您的新视频命名为 `promo.mp4`。
2. 将文件放入项目的 `public/` 文件夹下。
3. 如果文件名没变，无需修改代码；如果想用外部链接，修改以下位置：
```tsx
// 找到第 13 行左右
<source src="/promo.mp4" type="video/mp4" />
```

---

## 2. 更新“同款服装”网格
确保网格中的产品顺序与视频中出现的顺序一致。

**文件路径：** `src/components/ProductGrid.tsx`

**操作步骤：**
1. 找到顶部的 `PRODUCTS` 数组。
2. 修改 `name`（名称）、`price`（价格）和 `image`（图片链接）。
3. 如果使用本地图片，将图片存入 `public/products/` 并在代码中引用路径（如 `/products/dress1.jpg`）。

---

## 3. SEO 博客（长尾流量承接）
每当有一个爆款视频，建议发布一篇配套文章。

**文件路径：** `src/app/blog/page.tsx` (列表) 和文章对应的 Slug。

---

## 💡 运营小贴士
* **视频压缩：** 移动端视频建议控制在 5MB 以内，以保证极速打开。
* **图片比例：** 统一使用 3:4 的人像比例图片，视觉效果最整齐。
* **Buy Now 链接：** 确保按钮直接指向您的结账页或私域咨询工具。
